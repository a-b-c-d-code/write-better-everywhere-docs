Privacy Policy for Write Better Everywhere

Last Updated: January 2025

Privacy Summary: Write Better Everywhere is designed with privacy as a top priority. Your API keys are stored locally in your browser, your text is sent only to the AI provider you choose, and we never collect, store, or transmit your data to any third-party servers.

1. Information Collection and Use

1.1 Data Stored Locally

Write Better Everywhere stores the following information locally in your browser using Chrome's chrome.storage.sync API:

  • API Keys: Your API keys for Anthropic Claude, OpenAI, Google Gemini, or custom API endpoints (if you choose to configure them)
  • Settings: Your preferences including:
    • Selected AI provider (Anthropic, OpenAI, Gemini, Custom, or Demo mode)
    • Selected AI model
    • Extension enabled/disabled status
    • Trigger preferences (sentence completion, paragraph completion, fallback timer)
    • Minimum text length threshold
    • Floating button visibility preference

Important: This data is stored securely in your browser and synced across your Chrome browsers if you're signed into Chrome. It is never transmitted to us or any third party.

1.2 Data Transmitted to AI Providers

When you use Write Better Everywhere to analyze text, the extension sends:

  • Your text content: The text you want analyzed is sent directly from your browser to the AI provider you selected (Anthropic, OpenAI, Google Gemini, or your custom API)
  • Your API key: Used to authenticate with the AI provider

Important: This data goes directly from your browser to the AI provider's servers. Write Better Everywhere acts as a messenger and does not intercept, store, or process this data on any intermediate servers.

1.3 Demo Mode

When using Demo Mode (no API key configured), all text analysis happens locally in your browser using basic pattern matching. No data is transmitted to any external servers.

2. Third-Party AI Provider Privacy Policies

When you use Write Better Everywhere with an AI provider, that provider's privacy policy governs how they handle your data. Please review the privacy policy of your chosen provider:

  • Anthropic Claude: https://www.anthropic.com/privacy
  • OpenAI: https://openai.com/privacy
  • Google Gemini: https://ai.google.dev/gemini-api/terms

Note: Each provider has different data retention and usage policies. We recommend reviewing their policies before using their services.

3. Permissions Explained

Write Better Everywhere requires the following Chrome extension permissions:

  • storage: To save your API keys and settings locally in your browser
  • <all_urls> (access to all websites): To work on any website where you write text (Gmail, Twitter, Google Docs, etc.) and to access text content when you request analysis
  • contextMenus: To provide the right-click "Analyze with Write Better Everywhere" option in text fields

Important: These permissions are used solely to provide the extension's functionality. We do not use them to collect data, track your browsing, or perform any actions without your explicit request.

4. Data We Do NOT Collect

Write Better Everywhere does not:

  • Collect or store any user data on external servers
  • Track your browsing history or behavior
  • Use analytics or telemetry services
  • Share your data with any third parties (except the AI provider you explicitly choose)
  • Contain advertisements or third-party tracking scripts
  • Automatically analyze your text without your request

5. Your Control

You have complete control over your data:

  • When data is sent: Text is only sent for analysis when you explicitly click "Analyze" or use a manual trigger
  • Which provider receives data: You choose which AI provider to use (or use Demo mode with no external transmission)
  • Your API keys: You manage your own API keys and can delete them at any time through the extension settings
  • Extension removal: Uninstalling the extension removes all stored settings and API keys from your browser

6. Security

We take security seriously:

  • API keys are stored using Chrome's secure storage API with encryption
  • All API communications use HTTPS
  • The extension is open source - you can review the code at GitHub
  • No third-party dependencies for core functionality (only AI provider APIs you explicitly configure)

7. Children's Privacy

Write Better Everywhere does not knowingly collect any information from children. The extension is designed for general use and does not target children specifically. Users must be old enough to create accounts with third-party AI providers if they choose to use those services.

8. Changes to This Privacy Policy

  • Update the "Last Updated" date at the top of this policy
  • Publish the new policy at this URL
  • Include a notice in the extension update notes if changes are significant

Write Better Everywhere complies with applicable data protection laws including GDPR and CCPA by:

  • Not collecting personal data without explicit user action
